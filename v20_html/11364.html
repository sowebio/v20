222        function Last_Insert_RowID return Integer_64;
223        --  The function usually returns the rowid of the most recent successful
224        --  INSERT into a rowid table or virtual table. Inserts into WITHOUT ROWID
225        --  tables are not recorded. If no successful INSERTs into rowid tables
226        --  have ever occurred on the database, then the function returns zero.
227        --  Each entry in most SQLite tables (except for WITHOUT ROWID tables) has
228        --  a unique 64-bit signed integer key called the "rowid". The rowid is
229        --  always available as an undeclared column named ROWID, OID, or _ROWID_
230        --  as long as those names are not also used by explicitly declared
231        --  columns. If the table has a column of type INTEGER PRIMARY KEY then
232        --  that column is another alias for the rowid (this text comes from
233        --  https://www.sqlite.org/c3ref/last_insert_rowid.html).
